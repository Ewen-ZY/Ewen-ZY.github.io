windows10中使用小飞机。

介绍

Shadowsocks-windows，个人唯一会用的 windows 下的工具，俗称”小飞机“。此文旨在介绍小飞机的各项按钮。

依赖

需要安装 .NET Framework 4.6.2 和 Microsoft Visual C++ 2015 Redistributable (x86)。如果安装了这些还不能使用，请查看小飞机官网安装最新版本的依赖。

下载

进入官网下载界面下载最新版本（latest release）的小飞机。

解压

将下载好的小飞机解压放到你想放置的地方。

Note：强烈推荐放置非C盘的安装软件的地方。

启动

解压后双击启动小飞机，可以在托盘部分找到小飞机。

系统代理

选择代理模式，

  1. 禁用：不使用小飞机。
  2. PAC模式：针对大部分常用的需要翻*的网站，前人都已经摸索出来了，因此这个模式是最常使用的模式。它可以识别诸如Google等需要翻*的网站自动翻出去，识别百度等不需要翻*的国内网站直连。
  3. 全局模式：在PAC模式中，对于翻*速度有点慢可以暂时使用一下。

服务器

选择你想要用的服务器，如果没有服务器，那么请点击“编辑服务器…”。

一般只需要填写“服务器地址”、“服务器端口”、“密码”、“加密”等，点击“确定”。详细自行学习。

如果有人想分享节点给你，那么可以使用“扫描屏幕中的二维码…”、“从剪贴板导入…”按钮。

PAC

一般情况下，使用本地 PAC 和保护本地 PAC 就好了，小飞机使用一段时间后，建议点击“从GFWList更新本地PAC”更新一下要走代理的网站列表。

如果想把一些国内可以访问但访问速度很慢的网站设置为走代理，那么可以点击编辑本地PAC文件，编辑user-rule.txt。编辑规则传送门。

开机启动

推荐使用开机自动启动。如果你的软件放在C盘，可能在自动启动时还需要点击“确定”，所以不推荐放在C盘。
